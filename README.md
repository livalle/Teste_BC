# Teste_BC
Desafio técnico para analista de dados 
## Descrição do Projeto

Este repositório contém um desafio técnico destinado a candidatos à vaga de analista de dados. O objetivo é avaliar suas habilidades em manipulação de dados, análise e visualização.

## Objetivos do Desafio

1. **Análise de Dados**: Realizar uma análise exploratória dos dados fornecidos.
2. **Limpeza de Dados**: Identificar e corrigir problemas de qualidade nos dados.
3. **Visualização**: Criar visualizações que ajudem a interpretar os dados de forma clara e concisa.
4. **Relatório**: Elaborar um relatório que resuma suas descobertas e insights.

## Dados Fornecidos
Planilhas em CSV, XlS e web scrapping
